# 全村的希望 · 微信小游戏上架指南

## 📋 上架前检查清单

- [x] AppID 已配置：`wxec56b6124ca18c16`
- [x] project.config.json 已填入真实 AppID
- [x] packOptions 已排除非游戏文件（推广方案、图标、预览页等）
- [ ] 图标已准备（用 icon-方案5-温暖家庭.png，需裁剪为正方形）
- [ ] 游戏在开发者工具中正常运行

---

## 第1步：准备图标

1. 打开 `icon-方案5-温暖家庭.png`
2. 确认是 1024×1024 的正方形 PNG
3. 如果尺寸不对，用 Mac 自带的「预览」裁剪调整

---

## 第2步：用微信开发者工具打开项目

1. 打开「微信开发者工具」
2. 点「导入项目」或「＋」
3. 目录选择：`/Users/tangxiaotao/WorkBuddy/Claw/quancun-game/`
4. AppID 选择：`wxec56b6124ca18c16`（全村的希望）
5. 点「确定」

⚠️ 如果提示项目已存在，直接从列表里点开就行

---

## 第3步：本地测试

1. 在模拟器里跑一遍游戏
2. 检查：开始画面 → 选事件 → 看结局 → 重新来过
3. 确认没有报错（控制台无红色错误）

---

## 第4步：上传代码

1. 点右上角「上传」按钮
2. 版本号填：`3.0.0`
3. 项目备注填：`温暖家庭风改版，人生模拟器`
4. 点「上传」，等进度条走完

---

## 第5步：去微信公众平台提交审核

1. 打开浏览器，登录 [mp.weixin.qq.com](https://mp.weixin.qq.com)
2. 左侧菜单 → 「版本管理」
3. 找到刚上传的「开发版本」
4. 点「提交审核」

### 填写审核信息：

| 字段 | 填什么 |
|------|--------|
| 游戏名称 | 全村的希望 |
| 游戏简介 | 从22岁到50岁，你的每一个选择都在改写人生。8种结局，你的人生会是什么样？ |
| 游戏类型 | 休闲 → 模拟 |
| 游戏引擎 | 原生Canvas |
| 隐私政策 | 游戏不收集任何用户数据（无网络请求、无登录、无广告） |

### 截图要求：
- 至少 2 张，最多 10 张
- 建议截：标题画面、事件选择画面、结局画面
- 在开发者工具模拟器里截图即可

5. 全部填完 → 点「提交」
6. 等审核（通常 1-3 个工作日）

---

## 第6步：审核通过后发布

1. 收到审核通过的通知
2. 回到 mp.weixin.qq.com → 版本管理
3. 找到「审核版本」→ 点「发布」
4. 游戏上线！🎉

---

## ⚠️ 审核注意事项

- 游戏内**无广告、无付费、无用户数据收集**，审核应该很顺利
- 事件文案虽有吐槽但**无政治敏感、无色情暴力**
- 结局文案**都是正能量或幽默调侃**，不会被打回
- 如果被打回，看驳回原因改对应事件文案即可
