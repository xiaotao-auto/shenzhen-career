---

## 📌 改前必读

1. 用任意文本编辑器打开 `data-config.js`（推荐 VS Code、记事本也行）
2. **只改冒号右边的数字**，左边的英文变量名不要动
3. 改完保存，上传到 GitHub 即可生效
4. 所有薪资单位：**元/月**（除非标注了 K，那就是千元/月）
5. 百分比数据写整数或小数都行，比如 `35` 或 `35.5`

---

## 📊 第1块：全局概览（首页大数字）

```
overview: {
  population: 2176,          // 常住人口（万）→ 去深圳统计局查最新
  youthRatio: 35,            // 35岁以下占比 % → 改百分比数字
  avgSalary: 10800,          // 青年月薪中位数 → 改金额
  netInflow: 48,             // 年净流入（万人）→ 改数字
  gradAvgSalary: 7200,       // 应届生起薪中位数 → 改金额
  salaryGrowth: 8.6,         // 薪资同比涨幅 % → 改百分比
  acceptBelow10k: 73,        // 愿接受1万以下比例 % → 改数字
  techP7Salary: 25000,       // 互联网P7中位数 → 改金额
  year: "2024",              // 数据年份 → 改年份字符串
  lastUpdate: "2024-12-01",  // 最后更新日期 → 改日期字符串
},
```

**举例**：2025年新数据出来了，青年月薪中位数涨到11500，就改：
```
avgSalary: 11500,
```


## 🏆 第7块：行业薪资排行

```
industrySalary: [
  { name: '🤖 AI / 人工智能',   salary: 24500, ratio: 1.00 },
  { name: '💻 互联网技术',       salary: 19500, ratio: 0.80 },
  ...
],
```

- `salary`：该行业中位数月薪（元）
- `ratio`：相对于AI行业的薪资比例（AI=1.00，互联网=0.80 表示是AI的80%）
- **改完 salary 后，ratio 也要自己算一下**：ratio = 该行 salary ÷ 第一行 salary

**举例**：AI涨到26000，互联网涨到21000：
```
{ name: '🤖 AI / 人工智能',   salary: 26000, ratio: 1.00 },
{ name: '💻 互联网技术',       salary: 21000, ratio: 0.81 },  // 21000÷26000≈0.81
```

## 🚀 第11块：就业赛道

```
careerTracks: [
  { icon: '🤖', name: 'AI / 大模型', heat: '超热门', salaryRange: '¥ 25K–50K', avgSalary: 36, demand: 95, jobs: 8.2 },
  ...
],
```

每个字段含义：
- `icon`：emoji图标（不用改）
- `name`：赛道名称
- `heat`：热度等级 → 可选 `'超热门'` / `'热门'` / `'稳定'` / `'冷门'`
- `salaryRange`：薪资范围显示文字 → 直接改字符串
- `avgSalary`：中位数月薪（**K**，36=3.6万/月）
- `demand`：需求指数 0-100
- `jobs`：岗位数量（万个）

**举例**：AI薪资涨了：`avgSalary: 40, salaryRange: '¥ 28K–55K'`

```
cityComparison: [
  { name: '深圳', avgSalary: 10800, gradSalary: 7200, rent: 3500, hukou: '较易', techJobs: 28.6, score: 92 },
  ...
],
```

- `avgSalary`：月薪中位数（元）
- `gradSalary`：应届生起薪中位数（元）
- `rent`：一居房租中位数（元/月）
- `hukou`：落户难度 → `'轻松'` / `'较易'` / `'较难'` / `'极难'`
- `techJobs`：科技岗占比 %
- `score`：综合评分 0-100

## ⚠️ 常见错误

| 错误 | 原因 | 修正 |
|------|------|------|
| 网页白屏 | 数字后面多写了逗号或少了逗号 | 检查每行末尾的逗号 |
| 图表不显示 | 改到了变量名（左边的英文） | 只改右边的数字 |
| 数据看着不对 | 百分比没加到100 | 检查所有百分比数组总和 |
